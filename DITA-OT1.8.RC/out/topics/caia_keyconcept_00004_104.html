Cash flow at risk (CFaR) is a more appropriate measure of risk for illiquid assets with long time horizons like private equity. A probability density function for terminal wealth dispersion for a portfolio of funds can be created using the total value to paid-in ratio, which is calculated using cash inflows and outflows as well as recent reported net asset values.
