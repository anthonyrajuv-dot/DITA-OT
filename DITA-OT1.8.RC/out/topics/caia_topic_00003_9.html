There are a couple of key points regarding the normal distribution and compounded returns. Assume we are compounding monthly returns to create a quarterly compounded return. If monthly returns are normally distributed, the quarterly returns using discrete compounding are not normally distributed. However, if monthly log returns are normally distributed, the quarterly log returns are also normally distributed. This is why log returns are often preferred for statistical analysis.
