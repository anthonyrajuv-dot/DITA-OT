Utilizing Style Boxes

LO 16.4: Demonstrate knowledge of real estate style boxes

For example:

  • Describe the basic characteristics of style boxes, and how they can be used in analyzing real estate investments.

A real estate style box offers another way to characterize real estate investments. An X is placed in the box that best describes the asset (or to locate a single manager or property based on the vertical and horizontal attributes). If percentages are located in each box, it indicates the percentage of the portfolio (or fund) invested in assets with those particular characteristics. Style boxes are also used to describe traditional assets, such as stocks and bonds. Figure 2 is an example of a bond style box. In this example, 30% of the portfolio is invested in short-term, low credit risk bonds.

Figure 3 illustrates a real estate style box with the NCREIF categories on the horizontal axis and the type of market on the vertical axis. Any attribute the analyst is interested in (e.g., life cycle, investment structure, or turnover) can be located on an axis. This type of box can help an investor better understand top-down allocation. The boxes may contain percentage allocations (as in Figure 2), expected returns, expected volatilities, expected gross returns, expected alphas, or some other attribute. Figure 3 illustrates a style box with expected returns located in each box. Notice that expected returns are lower for core properties in primary markets and higher for opportunistic properties in tertiary markets. This is exactly what you would expect given the risk/reward profiles of core real estate versus opportunistic real estate.
