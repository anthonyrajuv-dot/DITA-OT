LO 15.6

Real estate values obtained from transactions contain noise:

  • Noise classified as purely random error resulting from negotiated price.
  • Noise classified as temporal lag bias results due to the time lag between negotiation and reporting of a real estate transaction.

Real estate values obtained from appraisals also contain noise. Appraisal error is the appraiser’s estimation error and can be reduced by using more transaction data. The reduction in noise comes at a price of using older data and, hence, potentially increases the temporal lag bias. An appraisal that uses “n” more transactions would have of the original appraisal error.
