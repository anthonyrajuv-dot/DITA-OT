Commodity Index Return Types

LO 24.8: Demonstrate knowledge of commodity index return calculations.

For example:

  • Discuss four factors that influence returns on commodity indices, and their effects on return attribution.
  • Describe the four measures of return attribution (i.e., spot return, excess return, total return, and realized roll return) that are typically published by commodity index providers.
  • Calculate returns for a given commodity index.

Relative to equity indices, return attribution for commodity indices is complicated by the following factors:

  • Collateral. Differences in collateral investments can lead to differences in collateral returns.
  • Entry points. The timing of the rollover into new contracts influences returns.
  • Exit points. The exit timing from current contracts influences returns.
  • Weights. Commodity indices use different weighting schemes, which alters returns.

There are four key measures of returns for commodity indices. The return measures include:

  1. Excess return. Percentage change in the market value of futures contracts before adjustments are made for index changes.
  2. Spot return. Percentage change in the market value of futures contracts after adjustments are made for index changes.
  3. Realized roll return. Equals the excess return less the spot return.
  4. Total return. Equals the risk-free (i.e., collateral) return plus the excess return.

In the absence of index changes, the excess return and spot return will be equal and the roll return will be zero. As contracts are rolled (usually only a few days each month) or index weights are adjusted (generally once a year), the excess return and spot return will diverge and the roll return will be positive or negative.

Perhaps the best way to understand the return measures and their relationships to each other is to work through an example. Note that in the example, we use a multiplier, which represents the number of contracts for each commodity in the index.
