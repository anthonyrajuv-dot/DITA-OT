Continuous compounding refers to the continuous reinvestment of interest, in which case the simple return will equal where Rm→∞ is the continuously compounded return (i.e., the return upon which continuous compounding is applied). The continuously compounded return (i.e., log return) equals ln(1 + R).

Compounding in which interest is not continuously reinvested is known as discrete compounding.

The geometric mean return can be calculated using log returns: eM – 1, where M is the arithmetic average log return.
