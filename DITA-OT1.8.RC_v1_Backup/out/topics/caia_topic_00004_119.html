Benchmark Types

LO 8.4: Demonstrate knowledge of types of private equity benchmarks.

For example:

  • Define asset-based benchmarks and peer-group benchmarks.
  • Define absolute return and relative return in the context of benchmark categorization.

Two broad approaches to benchmark construction are the asset-based benchmark and peer-group-based benchmark. An asset-based benchmark, such as the S&P 500 Index, is built using public or private securities. A peer-group-based benchmark, such as the CISDM Equity Long/Short Index, is built using portfolios of investment managers.

Another way of categorizing returns relative to a benchmark involves determining whether a return is considered absolute or relative. An absolute return, which is typically how private equity is evaluated, is the return provided by a portfolio or individual asset over a given time period. A relative return represents the difference between the absolute return on an investment/portfolio and the returns on the investment/portfolio to which it is compared.
