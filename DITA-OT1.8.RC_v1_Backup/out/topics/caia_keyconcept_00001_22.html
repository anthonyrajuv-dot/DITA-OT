LO 4.5

Autocorrelation, illiquidity, and nonlinearity cause data to be non-normal. Autocorrelation refers to the correlation among lagged values of a random variable, which might cause outcomes to be more extreme than predicted by the normal distribution. Prices of illiquid assets are often estimated using appraisals, which frequently exhibit positive autocorrelation. Nonlinearities in returns often introduce skewness into the distribution.

The Jarque-Bera statistic is used to test data for departures from the normal distribution. The Jarque-Bera statistic tests the null hypothesis that the skewness and excess kurtosis for the data distribution jointly equal zero.
