LO 24.8

Relative to equity indices, return attribution for commodity indices is complicated by variability in collateral returns, entry points, exit points, and weights.

There are four key measures of returns for commodity indices:

  • Excess return—Percentage change in the market value of futures contracts before adjustments are made for index changes.
  • Spot return—Percentage change in the market value of futures contracts after adjustments are made for index changes.
  • Realized roll return—Equals the excess return less the spot return.
  • Total return—Equals the risk-free return plus the excess return.

In the absence of index changes, the excess return and spot return will be equal and the roll return will be zero. As contracts are rolled or index weights are adjusted, the excess return and spot return will diverge and the roll return will be positive or negative.
